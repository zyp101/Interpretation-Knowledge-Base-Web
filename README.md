# 安徽省主要农作物遥感解译知识库 · 展示平台（公开）

> 本仓库**仅用于展示平台界面与功能**，不含真实解译数据。

## ⚠ 关于数据

- 本公开站点使用**合成示例数据**（明确标注「【示例】」）驱动全部模块，仅作界面与功能演示。
- **真实解译数据（163 条解译规则 + 物候期 + 区县矢量）存于私有数据库**
  [`zyp101/Interpretation-Knowledge-Base`](https://github.com/zyp101/Interpretation-Knowledge-Base)，**未在此公开**。
- 如需完整数据，请向仓库所有者申请私有库访问权限，或将私有库中的 `index.html` 与 `data/` 一并部署到您自己的私有环境。

## 在线访问

通过 GitHub Pages 访问本仓库的 `index.html` 即可使用全部模块：

- 🗺️ **地图**：安徽省区县分布（边界矢量），点击区县查看示例记录
- 🔍 **知识库检索**：按季别 / 作物 / 县 / 波段筛选 + 全文搜索（示例数据）
- 🖼️ **影像图版**：7 种作物 × 243 假彩色 / 321 真彩色（AI 生成示意图）
- 📅 **物候期对照**：36 旬物候阶段时间线（示例物候期）
- ⚠️ **矛盾检测**：Error / Warn / Info 分级（示例条目）
