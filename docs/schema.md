# 字段详细字典

## 1. knowledge_base.json 结构

```json
{
  "meta": {
    "title": "安徽省主要农作物遥感解译知识库",
    "generated": "2026-07-07",
    "sources": { ... },
    "counts": { ... },
    "seasons": ["夏收", "早稻", "秋收"]
  },
  "entries": [ ... ],      // 目标作物解译条目数组
  "confuse": [ ... ],      // 易混淆地类对比条目
  "counties": { ... },     // 县代码 → {name, ds, city_code}
  "contradictions": [ ... ] // 矛盾/一致性检测结果
}
```

## 2. entries 条目字段

| JSON 键 | 中文名 | 类型 | 取值范围 | 示例 | 必填 |
|---|---|---|---|---|---|
| `season` | 季别 | string | `"夏收"` / `"早稻"` / `"秋收"` | `"夏收"` | ✅ |
| `county` | 县名称 | string | 安徽省区县名 | `"庐江县"` | ✅ |
| `county_code` | 县代码 | string | 6 位行政区划代码 | `"340124"` | ✅ |
| `crop` | 作物名称 | string | 冬小麦 / 油菜籽 / 早稻 / 中稻 / 夏玉米 / 大豆 / 花生 | `"冬小麦"` | ✅ |
| `phase` | 影像时相 | date (YYYY-MM-DD) | 推荐遥感影像采集日期 | `"2026-03-11"` | ✅ |
| `phase_md` | 时相月日 | array [month, day] | [3, 11] | `[3, 11]` | ✅（派生） |
| `band_false` | 假彩色波段 | string | `"243"` 或 `"432"` | `"432"` | ✅ |
| `band_true` | 真彩色波段 | string | `"321"` 或 `""` | `"321"` | 部分库有 |
| `feature` | 影像特征 | text | 该作物在指定波段下的光谱/纹理/形状描述 | `"呈红色调，纹理均匀..."` | 多数有 |
| `remark` | 备注 | text | 补充说明 | `""` | 可选 |
| `_pheno_stage` | 物候期关联 | string | 自动匹配：`"变种名:阶段"` 如 `"冬小麦:拔节期"` | `"冬小麦:拔节期"` | 派生 |

### band_false 取值详解

| 值 | 波段分配 | 典型用途 |
|---|---|---|
| `"243"` | Band 4(NIR), Band 3(Red), Band 2(Green) | 标准假彩色，健康植被亮红 |
| `"432"` | Band 4(NIR), Band 3(Red), Band 1(Blue) | 变体假彩色，增强植被-非植被对比 |

> 注：原始 Excel 中波段列标题写作「假彩色影像波段（2,4,3）（4,3,2）」，表示该列可取 243 或 432 两种组合。

## 4. counties 县注册表

键为县代码字符串（如 `"340124"`），值对象：

| 子键 | 说明 | 示例 |
|---|---|---|
| `name` | 区县名称 | `"庐江县"` |
| `ds` | 地级市名称 | `"合肥市"` |
| `city_code` | 地级市代码 | `"3401"` |

## 5. contradictions 矛盾检测记录

| 子键 | 说明 | 取值 |
|---|---|---|
| `type` | 矛盾类型 | `"物候期不符"` / `"物候期缺失"` / `"同县同作物多记录"` |
| `severity` | 严重程度 | `"error"` / `"warn"` / `"info"` |
| `season` | 所属季别 | `"夏收"` 等 |
| `county` | 县名称 | — |
| `crop` | 作物 | — |
| `phase` | 相关影像时相 | — |
| `detail` | 详细说明文字 | — |

### 矛盾类型判定规则

| type | severity | 判定逻辑 |
|---|---|---|
| **物候期不符** | error | 推荐影像时相的旬索引处，该作物在物候期表中存在但阶段值为空（可能已收获或未播种） |
| **物候期缺失** | warn | 该县+作物组合在物候期表中无任何变种记录（数据覆盖缺口） |
| **同县同作物多记录** | info/warn | 同一县同一作物出现 ≥2 条解译记录且波段或时相不完全一致 |

### 物候期匹配归一化规则

知识库作物名 → 物候期基准名映射：

| KB 作物名 | 匹配的物候期名模式 |
|---|---|
| `油菜籽` | `油菜`、`油菜1`、`油菜2`、`油菜3`、`油菜（直播）` |
| `冬小麦` | `冬小麦`、`冬小麦1..5` |
| `早稻` | `早稻`、`早稻(移栽)`、`早稻（直播）` |
| 其他 | 直接匹配（去尾部序号和括号注记） |

## 6. phenology.json 物候期数据结构

```
{
  "区县代码(字符串)": {
    "区县名称(字符串)": {
      "作物名(字符串)": [ 36 个元素数组 ]
      // 元素 i = 第 i 旬的物候阶段名称（字符串），空旬为 null
    }
  }
}
```

36 旬顺序：1月上旬(0)、1月中旬(1)、1月下旬(2)、…、12月下旬(35)

常见阶段值：播种期、出苗期、返青期、分蘖期、拔节期、孕穗期、抽穗期、开花期、灌浆期、成熟期、收获期。null 表示该旬不在作物的生育期内。
